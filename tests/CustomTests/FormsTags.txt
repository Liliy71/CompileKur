<h2>Элемент select</h2>

<p>Элемент select определяет раскрывающийся список:</p>

<form action="/action_page.php">
  <label for="cars">Выбери авто:</label>
  <select id="cars" name="cars">
    <option value="volvo">Вольво</option>
    <option value="saab">Сааб</option>
    <option value="fiat">Фиат</option>
    <option value="audi">Ауди</option>
  </select>
  <input type="submit">
</form>

<h2>Предварительно выбранный вариант</h2>

<p>Вы можете предварительно выбрать опцию с атрибутом selected:</p>

<form action="/action_page.php">
  <label for="cars">Выберите авто:</label>
  <select id="cars" name="cars">
    <option value="volvo">Вольво</option>
    <option value="saab">Сааб</option>
    <option value="fiat" selected>Фиат</option>
    <option value="audi">Ауди</option>
  </select>
  <input type="submit">
</form>

<h2>Видимые значения параметров</h2>

<p>Используйте атрибут size, чтобы указать количество видимых значений.</p>

<form action="/action_page.php">
  <label for="cars">Выберите авто:</label>
  <select id="cars" name="cars" size="3">
    <option value="volvo">Вольво</option>
    <option value="saab">Сааб</option>
    <option value="fiat">Фиат</option>
    <option value="audi">Ауди</option>
  </select><br><br>
  <input type="submit">
</form>

<h2>Разрешить множественный выбор</h2>

<p>Используйте множественный атрибут, чтобы позволить пользователю выбрать более одного значения.</p>

<form action="/action_page.php">
  <label for="cars">Выбери авто:</label>
  <select id="cars" name="cars" size="4" multiple>
    <option value="volvo">Вольво</option>
    <option value="saab">Сааб</option>
    <option value="fiat">Фиат</option>
    <option value="audi">Ауди</option>
  </select><br><br>
  <input type="submit">
</form>

<p>Нажмите и удерживайте кнопку Ctrl (windows) / Command (Mac), чтобы выбрать несколько параметров.</p>

<h2>Элемент textarea</h2>

<p>Элемент textarea определяет текстовое поле ввода.</p>

<form action="/action_page.php">
  <textarea name="message" rows="10" cols="30">Кошка играла в саду.</textarea>
  <br>
  <input type="submit">
</form>

<form action="../action_page.php">
  <textarea name="message" style="width:200px; height:600px;">Кошка играла в саду.</textarea>
  <br>
  <input type="submit">

<h2>Группировка данных формы с помощью набора полей</h2>

<p>Элемент fieldset используется для группировки связанных данных в форме, а элемент legend определяет заголовок элемента fieldset.</p>

<form action="/action_page.php">
  <fieldset>
    <legend>Персональная информация:</legend>
    <label for="fname">Имя:</label><br>
    <input type="text" id="fname" name="fname" value="Андрей"><br>
    <label for="lname">Фамилия:</label><br>
    <input type="text" id="lname" name="lname" value="Щипунов"><br><br>
    <input type="submit" value="Submit">
  </fieldset>
</form>

<h2>Элемент datalist</h2>

<p>Элемент datalist задает список предопределенных параметров для элемента ввода.</p>

<form action="/action_page.php">
  <input list="browsers" name="browser">
  <datalist id="browsers">
    <option value="Edge">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist>
  <input type="submit">
</form>

<h2>Элемент output</h2>
<p>Элемент output представляет результат вычисления.</p>

<form action="/action_page.php"
oninput="x.value=parseInt(a.value)+parseInt(b.value)">
  0
  <input type="range" id="a" name="a" value="50">
  100 +
  <input type="number" id="b" name="b" value="50">
  =
  <output name="x" for="a b"></output>
  <br><br>
  <input type="submit">
</form>
