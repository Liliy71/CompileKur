<h2>Поле text</h2>
<p><strong>input type="text"</strong> определяет однострочное поле ввода текста:</p>

<form action="/action_page.php">
  <label for="fname">Имя:</label><br>
  <input type="text" id="fname" name="fname"><br>
  <label for="lname">Фамилия:</label><br>
  <input type="text" id="lname" name="lname"><br><br>
  <input type="submit" value="Отправить">
</form>

<p>Обратите внимание, что сама форма не видна.</p>
<p>Также обратите внимание, что ширина текстового поля по умолчанию составляет 20 символов.</p>

<h2>Поле password</h2>

<p><strong>input type="password"</strong> определяет поле пароля:</p>

<form action="/action_page.php">
  <label for="username">Имя пользователя:</label><br>
  <input type="text" id="username" name="username"><br>
  <label for="pwd">Пароль:</label><br>
  <input type="password" id="pwd" name="pwd"><br><br>
  <input type="submit" value="Отправить">
</form>

<p>Символы в поле пароля замаскированы (отображаются звездочками или кружками).</p>

<h2>Кнопка submit</h2>

<p><strong>input type="submit"</strong> определяет кнопку для отправки данных формы обработчику формы:</p>

<form action="/action_page.php">
  <label for="fname">Имя:</label><br>
  <input type="text" id="fname" name="fname" value="Андрей"><br>
  <label for="lname">Фамилия:</label><br>
  <input type="text" id="lname" name="lname" value="Щипунов"><br><br>
  <input type="submit" value="Отправить">
</form> 

<p>Если вы нажмете "Отправить", данные формы будут отправлены на страницу с именем  "/action_page.php".</p>

<form action="/action_page.php">
  <label for="fname">Имя:</label><br>
  <input type="text" id="fname" name="fname" value="Андрей"><br>
  <label for="lname">Фамилия:</label><br>
  <input type="text" id="lname" name="lname" value="Щипунов"><br><br>
  <input type="submit">
</form> 

<h2>Кнопка reset</h2>

<p><strong>input type="reset"</strong> определяет кнопку сброса, которая сбрасывает все значения формы к значениям по умолчанию:</p>

<form action="/action_page.php">
  <label for="fname">Имя:</label><br>
  <input type="text" id="fname" name="fname" value="Андрей"><br>
  <label for="lname">Фамилия:</label><br>
  <input type="text" id="lname" name="lname" value="Щипунов"><br><br>
  <input type="submit" value="Отправить">
  <input type="reset" value="Сброс">
</form> 

<p>Если вы измените входные значения и затем нажмете кнопку "Сброс", данные формы будут сброшены до значений по умолчанию.</p></p>

</form>

<h2>Поле number</h2>

<p><strong>input type="number"</strong> определяет числовое поле ввода.</p>

<p>Вы можете использовать атрибуты min и max, чтобы добавить числовые ограничения в поле ввода:</p>

<form action="/action_page.php">
  <label for="quantity">>Количество (от 1 до 5):</label>
  <input type="number" id="quantity" name="quantity" min="1" max="5">
  <input type="submit" value="Отправить">
</form>

<h2>Поле number</h2>

<p>В зависимости от поддержки браузера: в поле ввода будут применяться фиксированные шаги.</p>

<form action="/action_page.php">
  <label for="quantity">Количество:</label>
  <input type="number" id="quantity" name="quantity" min="0" max="100" step="10" value="30">
  <input type="submit" value="Отправить">
</form>
